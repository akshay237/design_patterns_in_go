### Abstract Factory
```
Abstract Factory is a creational design pattern that lets you produce families of related objects without specifying their concrete classes.
It is used when we need to create multiple families of similar products.
```

### Pizza Store
```
- Implement a pizza store which provides pizza and garlic bread
- There are two stores for pizza that are dominos and pizza hut.
- We have to keep both the brands seprate and serve the customer request on what we want.
- We have an interface for pizza and garlic bread that will be implemented by both of the brands.
```
